Qui sommes nous ?

Nous sommes 4 jeunes développeurs étudiants à l'Access Code School pour pouvoir vivre notre passion ! Dans le cadre de notre formation, nous avons eu pour projet de réaliser un site à partir de bases de données existantes. C'est ainsi qu'est né le projet "Date'Art" ! Les codeurs en question se nomment respectivement : Anthony, Kevin, Quentin et Boussad, tous motivés à en découdre avec la matrice.
